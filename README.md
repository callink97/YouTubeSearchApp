# YouTubeSearchApp


A simple Android app that lets you search for YouTube videos and display the results.
What the app does

You type a keyword and press Search
The app fetches videos from YouTube API
Results show in a list with title, channel, date, description, and thumbnail
You can sort results by relevance, date, view count, rating, or title
Shows a loading indicator while fetching
Handles errors like no internet, empty search, or no results

How to run

Download and unzip the project
Open it in Android Studio
Wait for Gradle sync to finish
Run on emulator or real device

Built with Java
Retrofit (networking)
Glide (image loading)
RecyclerView + Material Design
