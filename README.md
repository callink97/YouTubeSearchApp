# YouTube Search App (Java)

Android app to search YouTube videos using the YouTube Data API v3.

## Features
- Search by keyword (button or keyboard Enter)
- Sort by: Relevance, Date, View Count, Rating, Title *(extra credit)*
- Each result shows: Title, Channel, Publish Date, Description, Thumbnail (Glide)
- ProgressBar while loading
- Error handling: empty input, network failure, API quota, no results

## Stack
- Java (no Kotlin)
- Retrofit 2 + Gson (networking)
- Glide 4 (images)
- Material Components + ViewBinding
- RecyclerView

## Setup
1. Clone / unzip the project
2. Open in Android Studio
3. Wait for Gradle sync
4. Run on emulator or device (API 24+)

# YouTubeSearchApp


A simple Android app that lets you search for YouTube videos and display the results.
What the app does

You type a keyword and press Search
The app fetches videos from YouTube API
Results show in a list with title, channel, date, description, and thumbnail
You can sort results by relevance, date, view count, rating, or title
Shows a loading indicator while fetching
Handles errors like no internet, empty search, or no results

How to run

Download and unzip the project
Open it in Android Studio
Wait for Gradle sync to finish
Run on emulator or real device

Built with Java
Retrofit (networking)
Glide (image loading)
RecyclerView + Material Design
