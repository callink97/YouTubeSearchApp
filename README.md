# YouTube Search App (Java)

Android app to search YouTube videos using the YouTube Data API v3.

## Features
- Search by keyword (button or keyboard Enter)
- Sort by: Relevance, Date, View Count, Rating, Title *(extra credit)*
- Each result shows: Title, Channel, Publish Date, Description, Thumbnail (Glide)
- ProgressBar while loading
- Error handling: empty input, network failure, API quota, no results

## Stack
- Java (no Kotlin)
- Retrofit 2 + Gson (networking)
- Glide 4 (images)
- Material Components + ViewBinding
- RecyclerView

## Setup
1. Clone / unzip the project
2. Open in Android Studio
3. Wait for Gradle sync
4. Run on emulator or device (API 24+)
